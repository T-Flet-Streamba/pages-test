# vor-collabgpt-functions

Azure Function Microservice responsible for sending and receiving payloads from Azure OpenAI with additional data filtering and transforming based on the additional data provided (e.g. LMRs).

Canonical documentation for the repository lives under [`docs/`](docs/): start with [`docs/product/overview.md`](docs/product/overview.md) and [`docs/product/usage.md`](docs/product/usage.md).
