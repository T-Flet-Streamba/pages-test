# Integrations

Each integration lists **purpose**, **artifact type** (API, index, queue, etc.), and **where in this repo** the wiring lives (paths and key symbols).


| External system                  | Role                                                                                                                            | Where in repo                                                                                                                                                          |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Azure OpenAI**                 | LLM completions for LangGraph nodes (`ChatOpenAI` against Azure endpoint)                                                       | `config.py` → `llm`; `collabgpt_lg/bot.py` (`GraphLogisticsBot.llms`); `collabgpt_lg/graph.py`                                                                         |
| **VOR AI Search** (OAuth client) | Semantic / structured search over CHEVRON indexes (flights, road transport, global, etc.)                                       | `config.py` → `ai_search`; `collabgpt_lg/endpoints_base.py` (`AISearchBaseClass`); wrappers in `collabgpt_lg/endpoints.py`; upload via `shared/index_uploader.py`      |
| **VOR Search** (separate client) | OAuth client + `VOR_SEARCH_BASE_URL`; per-entity search paths (global, flights, road transport jobs, shipments, voyages) are **hardcoded** under `config.vor_search.url`. Used by `vor_global_search` and, for **Shell UK** / **ExxonMobilGuyana**, structured **`find_flights`**, **`find_voyages`**, and (Guyana only) **`find_shipments`** | `config.py` → `vor_search`; `collabgpt_lg/endpoints_base.py` (`VorSearchBaseClass`, `_query_pipeline`); `FlightsByDescriptionVorSearch`, `VoyagesByDescriptionVorSearch`, `ShipmentsVorSearch`, `VorGlobalSearch`, etc. in `collabgpt_lg/endpoints.py` |
| **VOR Customer API**             | Bearer-authenticated REST for containers, flights, MRs, shipments, work orders, voyage manifests, etc.                          | `config.py` → `customer_api`; `CustomerAPIBaseClass`; classes under `collabgpt_lg/endpoints.py`                                                                        |
| **Data Enhancer**                | OAuth-style username/password API for multiple orgs and entitites (CHEVRON dated queries, Guyana cargo events, CCU hires, etc.) | `config.py` → `data_enhancer`; `DataEnhancerAPIBaseClass`; `collabgpt_get_flights`, `collabgpt_get_road_transport_jobs`, bulk RTJ                                      |
| **Azure Cosmos DB**              | Voyage summaries and AIS-related reads; CHEVRON only for now                                                                    | `shared/cosmos.py` (`AISData`); `collabgpt_get_vessels_in_transit_by_location/__init__.py`                                                                             |
| **Redis**                        | Caches for subscriptions, logistics summary, CHEVRON priority freight keys                                                      | `config.py` → `redis`; `shared/redis_cache.py`; `collabgpt_get_priority_items/__init__.py`; subscription caches in `collabgpt_check_subscriptions/classes.py`          |
| **Azure Storage Queue**          | Subscription assessment work items                                                                                              | `config.py` → `azure_storage` (`SUBSCRIPTION_QUEUE`, `AzureWebJobsStorage`); `collabgpt_check_subscription_queue/__init__.py`                                          |
| **User Actions API**             | Active subscriptions, notification agent allow list, close subscription                                                         | `config.py` → `user_actions`; `shared/user_actions.py`                                                                                                                 |
| **LangFlow**                     | External agents: notification and subscription flows (URLs in config; env vars `LANGFLOW_*`)                                  | `config.py` → `langflow`; `LangFlowBaseClass`; `NotificationAgent`, subscription assessment in `collabgpt_check_subscriptions/functions.py`                            |
| **Flowise (retired)**            | The team **moved orchestration to LangFlow**; the old Flowise instance is gone. The PO overdue **chatflow was not ported** until it is needed again. The timer function `collabgpt_po_shipments_trigger` still contains **Flowise client code** but is **dormant** in practice. | `config.py` → `flowise`; `collabgpt_po_shipments_trigger/classes.py` |
| **Mixpanel**                     | Product analytics events                                                                                                        | `shared/mixpanel.py`; `collabgpt_lg/__init__.py`, `collabgpt_check_subscription_queue/__init__.py`                                                                     |
| **Slack**                        | Operational logging webhook                                                                                                     | `shared/slack.py` (`slack_logging` decorator on most function entrypoints)                                                                                             |
